Для просчёта столкновений с геометрией уровня будет генерироваться векторная модель воксельного мира.

По чанку, где находится игрок и соседним чанкам строится список границ блоков в определённой форме:
На боковых границах прозрачности между блоками строится отрезок в плоскости XY с определённой высотой.
На верхних\нижних границах прозрачности блоков строится шестиуоолькик а плоскости XY  с определённой высотой.

Меши геометрии для просчёта коллизий должен строить мир в своём потоке физики( исходя из позиции игрока ).
Мир отдаёт копию мешей игроку ( из своего потока ), когда расчёт меша будет закончен.
